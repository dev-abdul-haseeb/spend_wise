# SpendWise

A new Flutter project.

## Getting Started

1. Run required code given in firebase to add firebase to project
2. Run: cd android, ./gradlew signingReport to get the SHA keys to connect project
3. Add required packages and then you are good to go.

## Packages used:

1. firebase_core
2. firebase_auth
3. cloud_firestore
4. bloc
5. flutter_bloc
5. google_fonts
6. Equatable
7. shared_preferences
